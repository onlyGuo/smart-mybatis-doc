---
title: 自动同步数据库
date: 2025/11/21
---

在使用 Smart Mybatis 进行开发时, 我们可以启用自动同步数据库功能, 以便在应用启动时根据实体类的定义自动创建或更新数据库表结构。这对于快速开发和迭代非常有帮助, 可以减少手动维护数据库结构的工作量。
## 启用自动同步数据库
要启用自动同步数据库功能, 可以在配置文件中添加以下配置:

```yaml
spring:
  mybatis:
    smart:
      auto-sync-db: true
```
或者使用 Properties 格式:

```properties
spring.mybatis.smart.auto-sync-db=true
```
在上述示例中, 我们将 `auto-sync-db` 选项设置为 `true`, 以启用自动同步数据库功能。
这样, 当应用启动时, Smart Mybatis 会检查实体类与数据库表结构的差异, 并根据实体类的定义自动创建或更新相应的数据库表。

## 注意事项
- 启用自动同步数据库功能可能会对生产环境的数据库造成不可预期的更改, 请谨慎使用, 建议仅在开发阶段启用此功能。
